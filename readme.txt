== Description ==
Datarc is a modern masonry blog WordPress Theme. Its stylish design and beautiful typography makes it an ideal theme for your blog site. It features styles for the following post formats: standard, video, audio, and gallery. It is fully responsive and retina ready 
so It will always look great and crisp on any devices and screen sizes. It's built with clean and organized code which makes it very easy to customize.
